## Current Phase: Phase 1 — Design Tokens Studio

**Goal:** Build a fully functional visual editor for design tokens (colors, typography, spacing, shadows) with 250+ theme presets and accessibility tools.

**Target:** v0.1.0

| Task | Priority | Status | Notes |
|------|----------|--------|-------|
| Central State Module | High | Pending | Foundation for everything |
| Main Editor HTML/CSS | High | Pending | UI shell with panels |
| Color Token Controls | High | Pending | Core color editing |
| Color Harmony Engine | High | Pending | OKLCH palettes via Chroma.js |
| Typography Controls | High | Pending | Font family, size, weight |
| Theme Presets (250+) | High | Pending | Gogh + Catppuccin + Material |
| Export Engine | Medium | Pending | CSS, JSON, URL |
| Spacing & Shadows | Medium | Pending | Completes token set |
| Dark/Light Toggle | Medium | Pending | Preview mode switching |
| WCAG Dashboard | Medium | Pending | Accessibility checks |
| Keyboard Shortcuts | Low | Pending | Power user features |
